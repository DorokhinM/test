G class 2019 year

ГЛАВНАЯ МАШИНЫ G class

Гелендваген 2019 года выпуска

По двигателям пока официальной информации нет. По предварительным данным, основными для нового G-Class станут бензиновый V8 объемом 4,0 л и дизель V6 объемом 2,9 л. Также появится модификация с 3,0-литровым бензиновым турбомотором. Все двигатели будут агрегатироваться только с 9-ступенчатой автоматической трансмиссией, передающей крутящий момент в пропорции 40/60 в пользу задней оси.
